# O Projeto
O objetivo deste projeto é criar um sistema de irrigação automático utilizando um arduíno yun. Utilizaremos o Arduíno Yun pela facilidade de integração com o parse.com e para a criação do painel de controle.

# Tags
arduino yun, watering, home garden

# Funcionalidades
- Regar a horta quando a terra estiver seca.
- Registrar data, hora, temperatura e foto na hora da irrigação.
- Enviar notificação push para app quando não tiver mais água no reservatório.
- Contrução de um app para acessar os dados.

# Sensores, Atuadores e componentes
- Arduino Yun
- Bomba de água (Aquelas usadas em parabrisas de carro)
- WebCam
- Sensor de temperatura
- Sensor de umidade do solo
- Sensor de reservatório de água
- Relé 5v
- Resistor de 220ohm
- Transistor
- Fonte 5v 1000mA para Arduíno
- Fonte 4.5v 2500mA para bombinha de água
- Garrafão de 5L de água para servir de caixa dágua.

# Parse
Foi utilizado o Parse (https://parse.com) para salvar os registros.
